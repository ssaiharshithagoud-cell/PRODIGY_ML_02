# Customer Segmentation using K-Means Clustering

This project is part of my Machine Learning Internship at Prodigy InfoTech.

## Objective
To group customers based on their purchase behavior using K-Means clustering.

## Features Used
- Annual Income
- Spending Score

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Output
Customers are grouped into different clusters representing various spending behaviors.
